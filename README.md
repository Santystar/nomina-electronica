# API Nómina Electrónica

## Descripción
API REST desarrollada para la gestión de empleados, departamentos, asistencia y cálculo de nómina del proyecto formativo de nómina electrónica.

---

## Tecnologías utilizadas
- Node.js
- Express
- MySQL
- Thunder Client
- Git / GitHub
